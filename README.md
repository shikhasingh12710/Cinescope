CineScope

CineScope is a React-based movie discovery application that allows users to search for movies by title and instantly view detailed information such as posters, release year, genre, plot summary, and IMDb ratings. The application focuses on clean UI, fast search, and seamless API integration to enhance the movie browsing experience.
